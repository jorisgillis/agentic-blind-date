# Agentic Blind Date

This project is a web application that will be used during the welcome
of a meet-up on agentic AI. The goal is that is fun, tongue-in-cheek,
and an example of agentic AI in practice. 

The setting is [Blind Date](https://en.wikipedia.org/wiki/Blind_date)
but with the participants of the meet-up. Instead of romantic
connections, we are looking for connections based on technical
preferences and GitHub profiles. 

## Tech Stack

- Go programming language
- Server-side rendered, templated frontend with HTMX and Tailwind
- SQLite3 storage
- Mistral AI as LLM
